问题描述

网页Github进不去，尽管已经连接梯子。
在Cmd中ping github.com 百分比丢包。

解决方法

  1. 在目录 找到hosts文件，在最后加上两行github的IP
  2. 如何还是ping不通，192.30.255.112 可能是旧的 GitHub IP，GitHub 的 IP 经常变。在cmd中输入以下指令检查最新 IP： 替换到第二步中的地址
