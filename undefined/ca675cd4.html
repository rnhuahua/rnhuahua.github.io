GitдҪҝз”Ё

еҸ‘иЎЁдәҺ2024-12-22|жӣҙж–°дәҺ2024-12-22|ејҖеҸ‘е·Ҙе…·
|жҖ»еӯ—ж•°:909|йҳ…иҜ»ж—¶й•ҝ:3еҲҶй’ҹ|жөҸи§ҲйҮҸ:

Git жҳҜдёҖдёӘеҲҶеёғејҸзүҲжң¬жҺ§еҲ¶зі»з»ҹпјҢз”ұ Linus Torvalds дәҺ 2005 е№ҙејҖеҸ‘пјҢз”ЁдәҺй«ҳж•Ҳз®ЎзҗҶе’Ңи·ҹиёӘд»Јз Ғжӣҙж”№гҖӮ

еҹәжң¬жҰӮеҝөпјҡ

  • е·ҘдҪңеҢә (Working Directory)пјҡдҪ еҪ“еүҚжӯЈеңЁеӨ„зҗҶзҡ„йЎ№зӣ®зӣ®еҪ•гҖӮ
  • жҡӮеӯҳеҢә (Staging Area)пјҡз”ЁжқҘжҡӮеӯҳдҝ®ж”№зҡ„ең°ж–№пјҢйҖҡеёёз”Ё git add е°Ҷжӣҙж”№ж”ҫе…ҘгҖӮ
  • зүҲжң¬еә“ (Repository)пјҡеӯҳеӮЁйЎ№зӣ®еҺҶеҸІзүҲжң¬зҡ„ең°ж–№пјҢз”Ё git commit жҸҗдәӨжӣҙж”№гҖӮ
  • иҝңзЁӢд»“еә“ (Remote Repository)пјҡжүҳз®ЎеңЁжңҚеҠЎеҷЁдёҠзҡ„зүҲжң¬еә“пјҢжҜ”еҰӮ GitHubгҖҒGitLabгҖӮ

е®үиЈ…

Windowsзі»з»ҹ

  1. и®ҝй—® Gitе®ҳзҪ‘ дёӢиҪҪйҖӮз”ЁдәҺWindowsзҡ„е®үиЈ…зЁӢеәҸгҖӮ
  2. еҸҢеҮ»дёӢиҪҪзҡ„е®үиЈ…зЁӢеәҸпјҢжҢүз…§жҸҗзӨәиҝӣиЎҢе®үиЈ…гҖӮ
  3. е®үиЈ…е®ҢжҲҗеҗҺпјҢжү“ејҖе‘Ҫд»ӨжҸҗзӨәз¬ҰпјҢиҫ“е…Ҙ git --version йӘҢиҜҒе®үиЈ…жҳҜеҗҰжҲҗеҠҹгҖӮ

Linuxзі»з»ҹе®үиЈ…

еңЁLinuxзі»з»ҹдёҠе®үиЈ…GitеҸҜд»ҘдҪҝз”ЁеҢ…з®ЎзҗҶеҷЁпјҢдҫӢеҰӮaptпјҲйҖӮз”ЁдәҺDebian/UbuntuпјүжҲ–yumпјҲйҖӮз”ЁдәҺCentOS/RHELпјүгҖӮ

  1. дҪҝз”Ёaptе®үиЈ…GitпјҲDebian/Ubuntuпјү
  2. дҪҝз”Ёyumе®үиЈ…GitпјҲCentOS/RHELпјү
  3. йӘҢиҜҒе®үиЈ…

й…ҚзҪ®

е®үиЈ…е®ҢжҲҗеҗҺпјҢйңҖиҰҒиҝӣиЎҢдёҖдәӣеҹәжң¬й…ҚзҪ®пјҡ

  1. и®ҫзҪ®з”ЁжҲ·еҗҚе’ҢйӮ®з®ұең°еқҖпјҲз”ЁдәҺжҸҗдәӨи®°еҪ•пјү

  2. жҹҘзңӢй…ҚзҪ®дҝЎжҒҜ

  3. зј–иҫ‘й…ҚзҪ®ж–Үд»¶

й…ҚзҪ®SSHеҜҶй’Ҙ

дҪҝз”ЁSSHеҜҶй’ҘеҸҜд»ҘжҸҗй«ҳдёҺGitд»“еә“дәӨдә’зҡ„е®үе…ЁжҖ§гҖӮ

  1. з”ҹжҲҗSSHеҜҶй’Ҙ

    жҢүз…§жҸҗзӨәж“ҚдҪңпјҢе»әи®®дҪҝз”Ёй»ҳи®Өи·Ҝеҫ„пјҢе№¶и®ҫзҪ®дёҖдёӘе®үе…Ёзҡ„еҜҶз ҒзҹӯиҜӯгҖӮ

  2. ж·»еҠ SSHеҜҶй’ҘеҲ°ssh-agent

  3. е°ҶSSHе…¬й’Ҙж·»еҠ еҲ°Gitжүҳз®ЎжңҚеҠЎпјҲдҫӢеҰӮGitHubгҖҒGitLabпјү

    • еӨҚеҲ¶SSHе…¬й’ҘеҶ…е®№
    • зҷ»еҪ•еҲ°дҪ зҡ„Gitжүҳз®ЎжңҚеҠЎиҙҰжҲ·пјҢжүҫеҲ°ж·»еҠ SSHеҜҶй’Ҙзҡ„и®ҫзҪ®йЎөйқўпјҢе№¶е°ҶеӨҚеҲ¶зҡ„е…¬й’ҘеҶ…е®№зІҳиҙҙиҝӣеҺ»гҖӮ

Gitе·ҘдҪңжөҒзЁӢ

Gitзҡ„еҹәжң¬е·ҘдҪңжөҒзЁӢеҰӮдёӢпјҡ

  1. е…ӢйҡҶиҝңзЁӢд»“еә“еҲ°жң¬ең°

  2. еҲӣе»әж–°еҲҶж”Ҝе№¶еҲҮжҚўеҲ°ж–°еҲҶж”Ҝ

  3. ж·»еҠ ж–Үд»¶еҲ°жҡӮеӯҳеҢә

  4. жҸҗдәӨжӣҙж”№

  5. жҺЁйҖҒеҲҶж”ҜеҲ°иҝңзЁӢд»“еә“

  6. еҲӣе»ә Pull RequestпјҲPRпјү
    еңЁ GitHub жҲ–е…¶д»–жүҳз®Ўе№іеҸ°дёҠеҲӣе»ә Pull RequestпјҢйӮҖиҜ·еӣўйҳҹжҲҗе‘ҳиҝӣиЎҢд»Јз Ғе®ЎжҹҘгҖӮPR еҗҲе№¶еҗҺпјҢдҪ зҡ„жӣҙж”№е°ұдјҡеҗҲе№¶еҲ°дё»еҲҶж”ҜгҖӮ

  7. еҗҲе№¶жӣҙж”№
    еңЁ PR е®Ўж ёйҖҡиҝҮе№¶еҗҲе№¶еҗҺпјҢеҸҜд»Ҙе°ҶиҝңзЁӢд»“еә“зҡ„дё»еҲҶж”ҜеҗҲе№¶еҲ°жң¬ең°еҲҶж”Ҝпјҡ

  8. еҲ йҷӨеҲҶж”Ҝ
    еҰӮжһңдёҚеҶҚйңҖиҰҒж–°еҠҹиғҪеҲҶж”ҜпјҢеҸҜд»Ҙе°Ҷе…¶еҲ йҷӨпјҡ жҲ–иҖ…д»ҺиҝңзЁӢд»“еә“еҲ йҷӨеҲҶж”Ҝпјҡ

еёёз”Ёе‘Ҫд»Ө

д»ҘдёӢжҳҜдёҖдәӣеёёз”Ёзҡ„Gitе‘Ҫд»Өпјҡ

  1. еҲқе§ӢеҢ–дёҖдёӘж–°зҡ„Gitд»“еә“

  2. е…ӢйҡҶдёҖдёӘиҝңзЁӢд»“еә“

  3. жҹҘзңӢеҪ“еүҚд»“еә“зҡ„зҠ¶жҖҒ

  4. ж·»еҠ ж–Үд»¶еҲ°жҡӮеӯҳеҢә

  5. жҸҗдәӨжӣҙж”№

  6. жҹҘзңӢжҸҗдәӨеҺҶеҸІ

  7. еҲҮжҚўеҲҶж”Ҝ

  8. еҲӣе»әж–°еҲҶж”Ҝ

  9. еҗҲе№¶еҲҶж”Ҝ

  10. еҲ йҷӨеҲҶж”Ҝ

  11. жҺЁйҖҒеҲҶж”ҜеҲ°иҝңзЁӢд»“еә“

  12. жӢүеҸ–иҝңзЁӢд»“еә“зҡ„жӣҙж”№

  13. жҹҘзңӢиҝңзЁӢд»“еә“

  14. ж·»еҠ иҝңзЁӢд»“еә“

  15. з§»йҷӨиҝңзЁӢд»“еә“
