  1. lauch.jsonиҙҹиҙЈзҡ„жҳҜеҗҜеҠЁд»»еҠЎпјҢжү§иЎҢж–Үд»¶(еҸҜжү§иЎҢж–Үд»¶)
  2. tasks.jsonиҙҹиҙЈзҡ„жҳҜй…ҚзҪ®зӣёе…ід»»еҠЎгҖӮз®ҖеҚ•жқҘиҜҙе°ұжҳҜиҙҹиҙЈзј–иҜ‘й“ҫжҺҘз”ҹжҲҗеҸҜжү§иЎҢж–Үд»¶пјҢе…¶е®һе°ұжҳҜжү§иЎҢз»Ҳз«Ҝзҡ„зј–иҜ‘жҢҮд»Ө[g++ -g main.cpp -o main.o]гҖӮжүҖд»ҘеңЁжү§иЎҢlaunch.jsonж–Үд»¶д№ӢеүҚеҝ…йЎ»е…Ҳжү§иЎҢtasks.json
  3. launch.jsonе’Ңtasks.jsonйҖҡиҝҮlaunch.jsonдёӯзҡ„preLaunchTaskе…іиҒ”иө·жқҘгҖӮlaunch.jsonдёӯзҡ„preLaunchTaskжҳҜдёәдәҶеҗҜеҠЁtasks.jsonзҡ„пјҢжүҖд»ҘpreLaunchTaskеҜ№еә”зҡ„ж Үзӯҫеә”иҜҘдёҺtask.jsonдёҖиҮҙгҖӮ
  4. c_cpp_properties.jsonвҖ”зј–иҜ‘зҺҜеўғзӣёе…ізҡ„и®ҫзҪ®

    1.2.1.еҶ…зҪ®еҸҳйҮҸ

    1.2.2.tasks.json

      tasks.jsonж–Үд»¶зҡ„дҪңз”ЁжҳҜе‘ҠиҜү VS Code еҰӮдҪ•жһ„е»әпјҲзј–иҜ‘пјүзЁӢеәҸпјҢе°Ҷи°ғз”Ё g++зј–иҜ‘еҷЁд»Һжәҗд»Јз ҒеҲӣе»әдёҖдёӘеҸҜжү§иЎҢж–Үд»¶гҖӮ
      еңЁдёҠж–№зҡ„дё»иҸңеҚ•дёӯпјҢйҖүжӢ© з»Ҳз«Ҝ>й…ҚзҪ®й»ҳи®Өз”ҹжҲҗд»»еҠЎ пјҢе°ҶеҮәзҺ°дёҖдёӘдёӢжӢүиҸңеҚ•пјҢжҳҫзӨә C++ зј–иҜ‘еҷЁзҡ„еҗ„з§Қйў„е®ҡд№үзј–иҜ‘д»»еҠЎгҖӮйҖүжӢ© C/C++: g++ build active fileгҖӮ
      д№ҹеҸҜд»ҘиҮӘе·ұе»әз«Ӣж–Үд»¶пјҢзӣҙжҺҘеӨҚеҲ¶дёӢйқўеҶ…е®№гҖӮ иҜҙжҳҺпјҡ

    1.2.3.lunch.json

    launch.json ж–Үд»¶з”ЁдәҺеңЁ Visual Studio Code дёӯй…ҚзҪ®и°ғиҜ•еҷЁ
    д»Һдё»иҸңеҚ•дёӯпјҢйҖүжӢ© иҝҗиЎҢ > ж·»еҠ й…ҚзҪ®пјҢйҖүжӢ©C++ (GDB/LLDB)гҖӮ д№ӢеҗҺе°ҶзңӢеҲ°еұ•зӨәеҗ„з§Қйў„е®ҡд№үи°ғиҜ•й…ҚзҪ®зҡ„дёӢжӢүеҲ—иЎЁ,йҖүжӢ© g++ build and debug active fileгҖӮе°ҶиҮӘеҠЁз”ҹжҲҗlaunch.jsonж–Үд»¶гҖӮд№ҹеҸҜд»ҘиҮӘе·ұе»әз«Ӣж–Үд»¶ иҜҙжҳҺпјҡ

    1.2.4.c_cpp_properties.json

    c_cpp_properties.json ж–Үд»¶пјҢе…Ғи®ёдҪ жӣҙж”№и®ҫзҪ®пјҢдҫӢеҰӮзј–иҜ‘еҷЁзҡ„и·Ҝеҫ„гҖҒеҢ…еҗ«и·Ҝеҫ„гҖҒC++ ж ҮеҮҶпјҲй»ҳи®Өдёә C++17пјүпјҢзӯүзӯүгҖӮ еҸҜд»ҘйҖҡиҝҮд»Һе‘Ҫд»Өйқўжқҝ (Ctrl+Shift+P) иҝҗиЎҢе‘Ҫд»Ө C/C++: Edit Configurations (UI) жқҘжҹҘзңӢ C/C++ й…ҚзҪ® UI иҜҙжҳҺпјҡ
